************
Glossary
************

.. glossary::
    :sorted:

    Handler
        Managed component for which `Cowbird` provides some sort of interactive :term:`Operation`.

    Operation
        Sequence of execution steps that configures or sends requests to other components of the `bird-house`_ stack
        in order to produce an end result where those components, that normally don't interact directly together,
        will be able to obtain a combined and enhanced feature.
